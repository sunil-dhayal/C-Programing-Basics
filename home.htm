C PROGRAMMING

C was developed by Dennis Ritchie in 1972. It is a procedural programming language. It's a system programming language mainly developed to write operating system.C language suitable for operating system or compiler development because it works with both high level and low level languages.

DATA STRUCTURE

A data structure is a data organization, management, and storage format that enables efficient access and modification. In data structure we basically go through following points:-Array,linked list,tree,stack,queue,graph,matrix,heap,binary search tree,binary tree, minimum spanning tree and sorting techniques.
