In it we included programs that needed String Related Operations.

Examples(click on question to see the answer)

Note:First try to do it by yourself their after go for answer.

1. Without using any string.h library functions Write a program to Compares two strings without regard to case.
2. Without using any string.h library functions Write a program to finds first occurrence of a given character in a string.
3. Without using any string.h library functions Write a program to finds last occurrence of a given character in a string.
4. Without using any string.h library functions Write a program to finds first occurrence of a given sub string in another string.
5. Without using any string.h library functions Write a program to sets all characters of string to a given character.
6. Without using any string.h library functions Write a program to sets first n characters of a string to a given character.
7. Without using any string.h library functions Write a program to reverses a string.
8. Write a program to count number of words in a string.
9. Without using any string.h library functions Write a program to converts a string to lowercase.
10. Without using any string.h library functions Write a program to converts a string to uppercase.
11. Without using any string.h library functions Write a program to appends one string at the end of another.
12. Without using any string.h library functions Write a program to appends first n characters of a string at the end of another.
13. Without using any string.h library functions Write a program to copies a string into another.
14. Without using any string.h library functions Write a program to copies first n characters of one string into another.
15. Without using any string.h library functions Write a program to compares two strings character by character.
16. Without using any string.h library functions Write a program to compares first n characters of two strings.
17. Write a program to sort a set of names stored in an array in alphabetical order.
18. Write a program to delete all vowels from a sentence. Assume that the sentence is not more than 80 characters long.
19. Write a program that takes a set of names of individuals and abbreviates the first, middle and other names except the last name by their first letter.
20. Write a program to count the number of occurrences of any two vowels in succession in a line of text. For example, in the sentence "Pleases read this application and give me gratuity' such occurrences are ea, ea, ui.
